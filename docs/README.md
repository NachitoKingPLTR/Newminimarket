# docs

Put a screenshot of the running terminal here named `screenshot.png` — the main
README links to it. Easiest way: run `python -m streamlit run app.py`, then on macOS press
**Cmd+Shift+4** and drag a box around the browser window, or use **Cmd+Shift+5**
for a full-window capture. Save/rename it to `docs/screenshot.png`.
